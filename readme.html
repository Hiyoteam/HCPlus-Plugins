HC++ Plugin Requirements

Thank you for developing more features for HC++! But in order for these plugins not to defeat the original purpose of creating our plugin system (to provide a better chat experience), we do have some requirements for the plugins.

1. Not a malicious plugin.

As stated above, plugins are meant to provide users with a better chat experience, but harmful plugins will undoubtedly not fulfill that condition. It is to avoid harmful plugins that we have created this plugin marketplace with an audit system.

2. Negatively targeted plugins.

Negative targeting plugins refer to negative plugins that are created to target someone (e.g. a user who installs this plugin will automatically send spamming messages to someone, etc.), and such plugins do not benefit the community.

3. SFW (Safe-For-Work).

We want to maintain a certain level of formality here, so NSFW contents are not allowed.

4. No code obfuscation.

To keep the review efficient and accurate, please do not code obfuscate your plugin.

5. No plugins that violates the law.

It is illegal to use code that violates the law, and we do not allow such code to be used in our plugin system.

6. Accurate description.

Please write an accurate description for your plugin and we will also review the plugin to see if it does this and only this.

7. No plugins that violates the privacy.

It is illegal to use code that violates the privacy, and we do not allow such code to be used in our plugin system.



If you are happy to follow the above rules, then feel free to publish your plugin!
